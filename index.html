基于 FastAPI 的极速网页多模态提取引擎

OmniExtract 万象支持对网页进行高保真双重解析，具备智能切片聚类、多模态文字与表格提取。在离线环境下支持调用本地原生 OCR（Windows Native OCR、macOS Vision OCR）及本地 RapidOCR 模型，在联网环境下无缝降级接入 MinerU 云端多模态大模型。

快速使用指南

1. 添加待处理项目： 您可以点击选择文件、直接拖入本地文件（图片、PDF、Word等），也可以在链接输入框中输入待解析的网页 URL。甚至可以直接粘贴截图（Ctrl+V）！

2. 选择识别引擎： 在右侧配置面板中选择适合的 API 引擎。如配置有 MinerU Token，可选用云端大模型；离线时可无缝降级选用系统原生 OCR 引擎，免装任何重型包。

3. 设定解析格式： 通过勾选“表格”、“纯文本”或“数学公式”等偏好设定，精细控制文档内容的渲染形式。

4. 开始并监控提取： 点击“开始提取”按键，系统将显示每张切片的实时提取进度。如果需要，您可以通过“暂停”、“继续”或“终止”按键进行灵活的流程控制。

5. 浏览与下载结果： 解析完成后，您可以在“切片明细”中对比原图和提取文字，或在“合成结果预览”里阅读完整渲染排版的文档，并一键打包下载为 Word、PDF、Markdown 或 Excel 表格文件。
